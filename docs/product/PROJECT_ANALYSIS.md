## Visão da Stack
- O sistema utiliza uma combinação de Gemini CLI, Node.js (MCP) e Bash.
- O "Cérebro Digital" foi migrado com sucesso para o Obsidian (Markdown).

## Observações Arquiteturais
- A arquitetura é orientada a processos (SDD), não a uma aplicação tradicional de backend.
- A segurança é garantida por hooks de FSM (Finite State Machine).
