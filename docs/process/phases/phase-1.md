# 🔍 Fase 1: Descoberta e Contextualização (Context-Check)

## Objetivo
Entender profundamente o impacto técnico da tarefa e recuperar conhecimentos do "Cérebro" (Obsidian).

## Passo a Passo
1.  **Análise Técnica:** Invoque `@context-backend` e `@context-frontend` para realizar uma análise detalhada do impacto.
2.  **Busca no Obsidian:** OBRIGATÓRIO consultar `docs/LESSONS_LEARNED.md` e ADRs via `mcp_ObsidianBrain_search_notes` para evitar erros passados.
3.  **Gestão de Contexto:** Invoque a skill `sdo-context-manager` para consolidar o conhecimento.
4.  **Relatório:** Informe o que foi encontrado, riscos técnicos e dependências.
5.  **Hook:** Execute `bash scripts/hooks/pre-spec.sh`.

## 🛑 BLOCKER
- Você deve listar pelo menos 2 lições aprendidas ou referências do Obsidian no seu relatório.
