## Objetivo
Registrar o aprendizado no Obsidian e entregar o valor ao usuário.

## Passo a Passo
1.  **RAG Ingest (Cérebro):** OBRIGATÓRIO salvar lições aprendidas (erros superados, decisões técnicas, "gotchas") como uma nova nota atômica no diretório `docs/lessons/` seguindo o `LESSON_TEMPLATE.md`. Use tags apropriadas e conecte com arquivos relacionados via [[wikilinks]].
2.  **Cleanup:** Execute `bash scripts/tools/archive-done-issues.sh`.
3.  **Handoff:** Apresente a entrega final ao usuário para o User Acceptance Testing (UAT).
