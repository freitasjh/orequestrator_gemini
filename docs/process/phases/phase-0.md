# 🚀 Fase 0: Bootstrap e Análise

## Objetivo
Garantir que os fundamentos do projeto (configuração e arquitetura) estejam sólidos antes de iniciar qualquer desenvolvimento.

## Passo a Passo
1.  **Verificação:** Garantir que `docs/product/PROJECT_CONFIG.md` e `docs/product/ARCHITECTURE.md` estejam preenchidos e atualizados.
2.  **Mapeamento:** Se for um projeto legado ou desconhecido, execute o comando `/analyze-project` para entender a stack e estrutura.

## 🛑 BLOCKER
- Não aceite demandas sem stacks claras e aprovadas.
- A fase só termina quando a arquitetura estiver clara.
